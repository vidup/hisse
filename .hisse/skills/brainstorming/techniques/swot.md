# SWOT

Outil d'analyse stratégique qui évalue une idée sur 4 dimensions :
2 facteurs internes (Forces/Faiblesses) et 2 facteurs externes (Opportunités/Menaces).

## Quand l'utiliser

- Évaluer si une idée est viable
- Choisir entre plusieurs options
- Préparer un pitch ou une présentation
- Identifier où concentrer ses efforts

## La matrice

```
            │  POSITIF            │  NÉGATIF
────────────┼────────────────────┼────────────────────
            │                    │
  INTERNE   │   FORCES           │   FAIBLESSES
  (nous)    │   Ce qu'on fait    │   Ce qui nous
            │   bien             │   manque
            │                    │
────────────┼────────────────────┼────────────────────
            │                    │
  EXTERNE   │   OPPORTUNITÉS     │   MENACES
  (marché)  │   Ce qu'on peut    │   Ce qui peut
            │   exploiter        │   nous nuire
            │                    │
```

## Questions par quadrant

**Forces** : Que fait-on mieux que les autres ? Quelles ressources uniques ? Quelle expertise ? Quel avantage compétitif ?

**Faiblesses** : Que nous manque-t-il ? Où sont nos lacunes ? Que font mieux les concurrents ? Quelles ressources limitées ?

**Opportunités** : Quelles tendances favorables ? Quels besoins non satisfaits ? Quels changements techno/réglementaires ? Quels partenariats possibles ?

**Menaces** : Qui sont les concurrents ? Quels obstacles externes ? Quels risques marché ? Quelles évolutions défavorables ?

## Exemple

**Idée** : App de coaching sportif personnalisé par IA

| FORCES | FAIBLESSES |
|--------|-----------|
| Équipe tech ML solide | Pas d'expertise sport/santé |
| Coûts bas (full remote) | Pas de notoriété |
| Exécution rapide | Budget marketing limité |

| OPPORTUNITÉS | MENACES |
|-------------|---------|
| Boom fitness post-COVID | Concurrence féroce (Nike, Freeletics) |
| Démocratisation IA | Régulation santé potentielle |
| Marché wearables croissant | Fatigue des apps fitness |

**Actions dérivées** :
- Force + Opportunité → Exploiter l'expertise ML pour du coaching via wearables
- Force + Menace → Vitesse d'exécution pour se différencier avant saturation
- Faiblesse + Opportunité → Partenariat avec des coachs pour le contenu
- Faiblesse + Menace → Cibler B2B d'abord plutôt que le grand public

## Conseils

- Être honnête sur les faiblesses — elles révèlent les vrais risques
- Opportunités et menaces sont souvent les deux faces d'une même tendance
- Croiser les quadrants pour dériver des stratégies
- Revisiter régulièrement — les conditions changent
