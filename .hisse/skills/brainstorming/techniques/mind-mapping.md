# Mind Mapping

Technique de visualisation qui organise les idées de manière arborescente
autour d'un concept central.

## Quand l'utiliser

- Idées dispersées, besoin de les structurer
- Explorer les relations entre sous-sujets
- Couvrir un sujet de manière exhaustive
- Difficulté à organiser de manière linéaire

## Processus

1. **Placer** l'idée centrale au coeur
2. **Créer** les branches principales (thèmes clés)
3. **Ajouter** les sous-branches (du général au spécifique)
4. **Relier** les concepts entre branches si pertinent
5. **Enrichir** avec couleurs, icônes, ou marqueurs visuels

## Exemple

**Idée** : App de gestion de budget personnel

```
                        Budget App
                     /    |    |    \
                   /      |    |      \
              Suivi   Analyse  Objectifs  Alertes
              /  \     /  \      |        |
         Revenus Dépenses  Graphes  Épargne  Seuils
         Récurrence  Catégories  Tendances  Échéances
```

## Format texte (quand pas d'outil visuel)

```markdown
# Budget App
## Suivi
- Revenus (récurrence, sources)
- Dépenses (catégories, tags)
## Analyse
- Graphiques mensuels
- Tendances sur 6 mois
## Objectifs
- Épargne cible
- Échéances
## Alertes
- Seuils de dépenses
- Rappels d'échéances
```

## Conseils

- Commencer par un brainstorm libre AVANT d'organiser
- Garder la cohérence thématique dans chaque branche
- Les connexions inter-branches sont souvent des insights
- Itérer — la première version n'est jamais la bonne
- Utiliser des verbes pour les processus, des noms pour les concepts
